## 🧠 About Me

I'm **Upender Naik Kelavath**, an **AI/ML Engineer** with **4+ years** of experience building intelligent, production-ready systems — from cutting-edge **LLMs and GenAI pipelines** to robust **MLOps infrastructure**.

- 🤖 I specialise in **Large Language Models, RAG pipelines, and Agentic AI** using LangChain, CrewAI & LangGraph
- 🚀 I architect end-to-end **ML solutions on AWS, Azure & GCP** and deploy with Docker & Kubernetes
- 📊 I turn messy data into **actionable insights** using Power BI, Tableau & advanced data science
- 🧪 I believe in clean code — **TDD, A/B Testing & MLflow** are part of my daily toolkit
- 💡 Always experimenting at the intersection of **NLP, Computer Vision & Cloud-native AI**
